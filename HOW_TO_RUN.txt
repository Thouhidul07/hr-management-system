HR Management System - How to Run

Prerequisites
1. Install Node.js.
2. Make sure MySQL is installed and running locally.
3. Open the project folder:
   c:\Users\msi\Desktop\HR-management\hr-management-system

Backend setup and run
1. Open a terminal in the backend folder:
   cd backend
2. Install backend dependencies:
   npm install
3. Check the backend .env file and make sure DATABASE_URL is set to:
   mysql://root:@localhost:3306/hr_management_system
4. Start the backend server:
   npm start
5. Wait until you see:
   Database connected successfully
   Server is running on port 3000

Frontend setup and run
1. Open a second terminal in the frontend folder:
   cd frontend
2. Install frontend dependencies:
   npm install
3. Start the frontend app:
   npm start
4. If port 3000 is already in use, choose 3003 when prompted.
5. Open the app in your browser at the port shown in the terminal.

How to log in
1. Go to the login page in the frontend.
2. Use this seeded account:
   Email: admin@hrms.com
   Password: password123
3. Click Login.

Useful backend commands
1. Start backend:
   cd backend
   npm start
2. Seed the database again if needed:
   cd backend
   npm run db:seed

Troubleshooting
1. If login fails, confirm the backend is running first.
2. If the backend cannot connect to MySQL, verify MySQL is running and the DATABASE_URL password is correct.
3. If the frontend cannot reach the backend, make sure the backend is running on port 3000.
4. If the frontend asks to use another port, accept 3003.
